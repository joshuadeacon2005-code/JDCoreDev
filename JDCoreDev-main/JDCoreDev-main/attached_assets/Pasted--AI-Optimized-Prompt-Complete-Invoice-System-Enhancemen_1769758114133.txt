# AI-Optimized Prompt: Complete Invoice System Enhancement

## Overview
Enhance the existing client project management system with comprehensive invoice generation capabilities, payment settings management, and accounts department contact tracking. This includes:

1. ✅ Generate invoices from **three locations**: Client Detail page, Projects page, and Documents page
2. ✅ Support **both invoice types**: Development (milestone-based) and Hosting (monthly recurring)
3. ✅ **Accounts Department** contacts for invoice communications
4. ✅ **Payment Settings** management with multiple payment methods
5. ✅ **Clear USD currency** display throughout invoices
6. ✅ **Auto-save** generated invoices to documents table

```typescript
app.get("/api/admin/payment-settings", async (req, res) => {
  const settings = await db.select().from(paymentSettings).limit(1);
  if (settings.length === 0) {
    const [newSettings] = await db.insert(paymentSettings)
      .values({ defaultCurrency: "USD" })
      .returning();
    return res.json(newSettings);
  }
  res.json(settings[0]);
});
```

**PATCH `/api/admin/payment-settings`**
```typescript
app.patch("/api/admin/payment-settings", async (req, res) => {
  const data = req.body;
  const [updated] = await db.update(paymentSettings)
    .set({ ...data, updatedAt: new Date() })
    .where(eq(paymentSettings.id, 1))
    .returning();
  res.json(updated);
});
```

### B. Invoice Generation Endpoints

**POST `/api/admin/invoices/hosting`**

Request:
```typescript
{
  clientId: number;
  projectIds: number[];
  invoiceDate: string;
  dueDate: string;
}
```

Response:
```typescript
{
  invoiceUrl: string;
  invoiceNumber: string;
  totalAmount: number;
  documentId?: number;  // If saved to documents table
}
```

Logic:
1. Validate projectIds belong to clientId
2. Fetch client (including accounts dept), projects, hosting fees, payment settings
3. Generate invoice number: `HOST-{clientId}-{YYYYMM}-{sequential}`
4. Create .docx with JDCOREDEV branding
5. Save to storage
6. Optionally save to documents table
7. Return download URL

**POST `/api/admin/invoices/development`**

Request:
```typescript
{
  clientId: number;
  projectId: number;
  milestoneId: number;
  invoiceDate: string;
  dueDate: string;
}
```

Response: Same as hosting

Logic:
1. Validate milestone → project → client relationship
2. Fetch all data including payment settings
3. Generate invoice number: `{PROJECT_CODE}-{invoiceSeq}-M{milestoneNum}`
   - Example: `PC3D-001-M1`
4. Create .docx with milestone details, deliverables, payment schedule
5. Save and return

**New Component 1**: `HostingInvoiceGeneratorDialog.tsx`

```typescript
interface HostingInvoiceGeneratorDialogProps {
  open: boolean;
  onOpenChange: (open: boolean) => void;
  clientId?: number;  // Pre-selected client (optional)
  onSuccess?: (invoiceUrl: string) => void;
}

export function HostingInvoiceGeneratorDialog({ open, onOpenChange, clientId, onSuccess }: Props) {
  const [selectedClientId, setSelectedClientId] = useState(clientId || null);
  const [selectedProjectIds, setSelectedProjectIds] = useState<number[]>([]);
  const [invoiceDate, setInvoiceDate] = useState(new Date());
  const [dueDate, setDueDate] = useState(addDays(new Date(), 7));

  // Fetch clients if no clientId provided
  const { data: clients } = useQuery(["/api/admin/clients"]);
  
  // Fetch hosting projects for selected client
  const { data: projects } = useQuery(
    ["/api/admin/projects", { clientId: selectedClientId, status: "hosting" }],
    { enabled: !!selectedClientId }
  );

  // Calculate total
  const totalAmount = useMemo(() => {
    if (!projects) return 0;
    return projects
      .filter(p => selectedProjectIds.includes(p.id))
      .reduce((sum, p) => sum + (p.monthlyFeeCents || 0), 0);
  }, [projects, selectedProjectIds]);

  const generateMutation = useMutation({
    mutationFn: async () => {
      return apiRequest("POST", "/api/admin/invoices/hosting", {
        clientId: selectedClientId,
        projectIds: selectedProjectIds,
        invoiceDate: invoiceDate.toISOString(),
        dueDate: dueDate.toISOString(),
      });
    },
    onSuccess: (data) => {
      toast({ title: "Invoice generated successfully" });
      onSuccess?.(data.invoiceUrl);
      onOpenChange(false);
    },
  });

  return (
    <Dialog open={open} onOpenChange={onOpenChange}>
      <DialogContent className="max-w-2xl">
        <DialogHeader>
          <DialogTitle>Generate Hosting Invoice</DialogTitle>
        </DialogHeader>

        <div className="space-y-4">
          {!clientId && (
            <div>
              <Label>Client</Label>
              <Select value={selectedClientId?.toString()} onValueChange={(v) => setSelectedClientId(Number(v))}>
                <SelectTrigger>
                  <SelectValue placeholder="Select client..." />
                </SelectTrigger>
                <SelectContent>
                  {clients?.map(c => (
                    <SelectItem key={c.id} value={c.id.toString()}>{c.name}</SelectItem>
                  ))}
                </SelectContent>
              </Select>
            </div>
          )}

          {selectedClientId && projects && (
            <>
              <div>
                <Label>Select Projects to Invoice</Label>
                <div className="space-y-2 mt-2">
                  {projects.map(project => (
                    <div key={project.id} className="flex items-center gap-2 p-3 border rounded">
                      <Checkbox
                        checked={selectedProjectIds.includes(project.id)}
                        onCheckedChange={(checked) => {
                          if (checked) {
                            setSelectedProjectIds([...selectedProjectIds, project.id]);
                          } else {
                            setSelectedProjectIds(selectedProjectIds.filter(id => id !== project.id));
                          }
                        }}
                      />
                      <div className="flex-1">
                        <div className="font-medium">{project.name}</div>
                        <div className="text-sm text-muted-foreground">
                          ${(project.monthlyFeeCents / 100).toFixed(2)}/month
                        </div>
                      </div>
                    </div>
                  ))}
                </div>
              </div>

              <div className="grid grid-cols-2 gap-4">
                <div>
                  <Label>Invoice Date</Label>
                  <Input type="date" value={format(invoiceDate, "yyyy-MM-dd")} onChange={(e) => setInvoiceDate(new Date(e.target.value))} />
                </div>
                <div>
                  <Label>Due Date</Label>
                  <Input type="date" value={format(dueDate, "yyyy-MM-dd")} onChange={(e) => setDueDate(new Date(e.target.value))} />
                </div>
              </div>

              <div className="p-4 bg-muted rounded-lg">
                <div className="flex justify-between items-center">
                  <span className="font-semibold">Total Amount:</span>
                  <span className="text-2xl font-bold">${(totalAmount / 100).toFixed(2)} USD</span>
                </div>
              </div>
            </>
          )}
        </div>

        <DialogFooter>
          <Button variant="outline" onClick={() => onOpenChange(false)}>Cancel</Button>
          <Button 
            onClick={() => generateMutation.mutate()}
            disabled={!selectedClientId || selectedProjectIds.length === 0 || generateMutation.isPending}
          >
            {generateMutation.isPending ? "Generating..." : "Generate Invoice"}
          </Button>
        </DialogFooter>
      </DialogContent>
    </Dialog>
  );
}
```

## Part 9: HKD Currency Conversion Display

### Overview
All invoices should display approximate HKD (Hong Kong Dollar) equivalents alongside USD amounts to help clients understand the cost in their local currency.

### Exchange Rate Management

**In Payment Settings**:
- Add `usdToHkdRate` field (DECIMAL(10,4), default 7.8000)
- Editable in Payment Settings admin page
- Should be updated periodically to reflect current exchange rates
- Note to admin: "Update periodically to reflect current exchange rates"

### Python Implementation

**Helper Function**:
```python
def format_currency_with_hkd(amount_cents: int, exchange_rate: float) -> tuple:
    """
    Format amount in USD with HKD equivalent
    
    Args:
        amount_cents: Amount in cents (USD)
        exchange_rate: Current USD to HKD rate
    
    Returns:
        tuple: (usd_string, hkd_string, usd_value, hkd_value)
    """
    usd_amount = amount_cents / 100
    hkd_amount = usd_amount * exchange_rate
    
    return (
        f"${usd_amount:,.2f} USD",
        f"≈ HK${hkd_amount:,.2f}",
        usd_amount,
        hkd_amount
    )

# Usage in invoice generation
exchange_rate = payment_settings.get('usdToHkdRate', 7.8)
usd_str, hkd_str, usd_val, hkd_val = format_currency_with_hkd(amount_cents, exchange_rate)
```
