# AI-Optimized Prompt: Crypto Portfolio Tracker with Alerts & News Integration

## Context
I have a client project management web application with an admin dashboard. I want to integrate a cryptocurrency tracking system that monitors coin prices, sends alerts via SMS/WhatsApp when certain conditions are met, and aggregates relevant news articles about tracked cryptocurrencies.

## Current Tech Stack
- **Frontend**: React with TypeScript
- **UI Components**: shadcn/ui (Card, Button, Dialog, Badge, Tabs, etc.)
- **State Management**: TanStack Query (React Query)
- **Backend**: Express.js with TypeScript
- **Database**: PostgreSQL with Drizzle ORM
- **Date Handling**: date-fns
- **Background Jobs**: Node.js (consider node-cron or similar)

## Project Goals

### Core Features:
1. 📊 **Cryptocurrency Price Tracking** - Monitor real-time prices for selected coins
2. 🔔 **Smart Alerts** - Price threshold alerts and percentage change notifications
3. 📱 **Multi-Channel Notifications** - SMS and/or WhatsApp via Twilio
4. 📰 **News Aggregation** - Scan and display relevant articles about tracked coins
5. 📈 **Portfolio Dashboard** - Visual dashboard with price charts and alerts

### B. Price Alerts Table

```sql

CREATE TYPE alert_status AS ENUM ('active', 'triggered', 'disabled');

CREATE TABLE price_alerts (
  id SERIAL PRIMARY KEY,

  -- Alert State

  trigger_price DECIMAL(20, 8),
  
  -- Notification Settings

  -- Metadata
  label VARCHAR(255),                          -- User-friendly label
  notes TEXT,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

CREATE INDEX idx_price_alerts_active ON price_alerts(status) WHERE status = 'active';
```

## Part 2: External API Integrations

### A. CoinGecko API (Free Tier)

**API Documentation**: https://www.coingecko.com/en/api/documentation

**No API Key Required** for basic usage (50 calls/minute)

**Key Endpoints**:

1. **Search Coins**: `GET /api/v3/search?query={query}`
2. **Coin Price**: `GET /api/v3/simple/price?ids={coin_ids}&vs_currencies=usd,hkd&include_market_cap=true&include_24hr_vol=true&include_24hr_change=true`
3. **Coin Details**: `GET /api/v3/coins/{coin_id}`
4. **Price History**: `GET /api/v3/coins/{coin_id}/market_chart?vs_currency=usd&days=7`

**Implementation** (`server/services/coingecko.ts`):

```typescript
import axios from 'axios';

const COINGECKO_BASE_URL = 'https://api.coingecko.com/api/v3';

export interface CoinPrice {
  coinId: string;
  priceUsd: number;
  priceHkd: number;
  marketCap: number;
  volume24h: number;
  percentChange24h: number;
}

export async function searchCoins(query: string) {
  const response = await axios.get(`${COINGECKO_BASE_URL}/search`, {
    params: { query }
  });
  return response.data.coins;
}

export async function getCoinPrices(coinIds: string[]): Promise<CoinPrice[]> {
  const response = await axios.get(`${COINGECKO_BASE_URL}/simple/price`, {
    params: {
      ids: coinIds.join(','),
      vs_currencies: 'usd,hkd',
      include_market_cap: true,
      include_24hr_vol: true,
      include_24hr_change: true,
    }
  });
  
  return Object.entries(response.data).map(([coinId, data]: [string, any]) => ({
    coinId,
    priceUsd: data.usd,
    priceHkd: data.hkd,
    marketCap: data.usd_market_cap,
    volume24h: data.usd_24h_vol,
    percentChange24h: data.usd_24h_change,
  }));
}

export async function getCoinDetails(coinId: string) {
  const response = await axios.get(`${COINGECKO_BASE_URL}/coins/${coinId}`);
  return {
    id: response.data.id,
    symbol: response.data.symbol.toUpperCase(),
    name: response.data.name,
    iconUrl: response.data.image?.large || response.data.image?.small,
    description: response.data.description?.en,
  };
}
```

**Recommended: CryptoPanic API** (`server/services/cryptonews.ts`):

```typescript
import axios from 'axios';

const CRYPTOPANIC_API_KEY = process.env.CRYPTOPANIC_API_KEY;
const CRYPTOPANIC_BASE_URL = 'https://cryptopanic.com/api/v1';

export interface NewsArticle {
  title: string;
  description: string;
  url: string;
  sourceName: string;
  publishedAt: Date;
  imageUrl?: string;
}

export async function fetchCoinNews(coinSymbol: string, limit: number = 10): Promise<NewsArticle[]> {
  try {
    const response = await axios.get(`${CRYPTOPANIC_BASE_URL}/posts/`, {
      params: {
        auth_token: CRYPTOPANIC_API_KEY,
        currencies: coinSymbol,
        kind: 'news',
        limit,
      }
    });
    
    return response.data.results.map((article: any) => ({
      title: article.title,
      description: article.title, // CryptoPanic doesn't provide description
      url: article.url,
      sourceName: article.source.title,
      publishedAt: new Date(article.published_at),
      imageUrl: article.metadata?.image,
    }));
  } catch (error) {
    console.error('Error fetching news:', error);
    return [];
  }
}
```

### C. Twilio Integration (SMS & WhatsApp)

**Setup**:
1. Sign up at https://www.twilio.com/
2. Get Account SID, Auth Token, Phone Number
3. Enable WhatsApp sandbox or get approved number

**Implementation** (`server/services/twilio.ts`):

```typescript
import twilio from 'twilio';
import { db } from '../db';
import { notificationSettings } from '@shared/schema';
import { eq } from 'drizzle-orm';

let twilioClient: any = null;

async function getTwilioClient() {
  if (twilioClient) return twilioClient;
  
  const settings = await db.select().from(notificationSettings).limit(1);
  if (!settings.length || !settings[0].twilioAccountSid) {
    throw new Error('Twilio not configured');
  }
  
  const config = settings[0];
  twilioClient = twilio(config.twilioAccountSid!, config.twilioAuthToken!);
  return twilioClient;
}

export async function sendSMS(message: string) {
  const client = await getTwilioClient();
  const settings = await db.select().from(notificationSettings).limit(1);
  const config = settings[0];
  
  if (!config.enableSms) {
    console.log('SMS disabled');
    return;
  }
  
  // Check quiet hours
  if (isQuietHours(config)) {
    console.log('Quiet hours - SMS not sent');
    return;
  }
  
  await client.messages.create({
    body: message,
    from: config.twilioPhoneNumber,
    to: config.recipientPhoneNumber,
  });
}

export async function sendWhatsApp(message: string) {
  const client = await getTwilioClient();
  const settings = await db.select().from(notificationSettings).limit(1);
  const config = settings[0];
  
  if (!config.enableWhatsapp) {
    console.log('WhatsApp disabled');
    return;
  }
  
  if (isQuietHours(config)) {
    console.log('Quiet hours - WhatsApp not sent');
    return;
  }
  
  await client.messages.create({
    body: message,
    from: `whatsapp:${config.twilioWhatsappNumber}`,
    to: `whatsapp:${config.recipientWhatsappNumber}`,
  });
}

function isQuietHours(config: any): boolean {
  if (!config.quietHoursStart || !config.quietHoursEnd) return false;
  
  const now = new Date();
  const currentTime = now.getHours() * 60 + now.getMinutes();
  
  const [startHour, startMin] = config.quietHoursStart.split(':').map(Number);
  const [endHour, endMin] = config.quietHoursEnd.split(':').map(Number);
  
  const startTime = startHour * 60 + startMin;
  const endTime = endHour * 60 + endMin;
  
  if (startTime < endTime) {
    return currentTime >= startTime && currentTime <= endTime;
  } else {
    return currentTime >= startTime || currentTime <= endTime;
  }
}
```

## Part 3: Background Jobs / Price Monitoring

**Install Dependencies**:
```bash
npm install node-cron
```

**Price Monitor Service** (`server/services/priceMonitor.ts`):

```typescript
import cron from 'node-cron';
import { db } from '../db';
import { trackedCoins, priceAlerts, priceHistory } from '@shared/schema';
import { eq, and } from 'drizzle-orm';
import { getCoinPrices } from './coingecko';
import { sendSMS, sendWhatsApp } from './twilio';

export function startPriceMonitoring() {
  // Run every 5 minutes
  cron.schedule('*/5 * * * *', async () => {
    console.log('Running price check...');
    await checkPrices();
  });
  
  console.log('Price monitoring started');
}

async function checkPrices() {
  try {
    // Get all active tracked coins
    const coins = await db.select()
      .from(trackedCoins)
      .where(eq(trackedCoins.isActive, true));
    
    if (coins.length === 0) return;
    
    // Fetch current prices
    const coinIds = coins.map(c => c.coinId);
    const prices = await getCoinPrices(coinIds);
    
    // Save to price history
    for (const price of prices) {
      await db.insert(priceHistory).values({
        coinId: price.coinId,
        priceUsd: price.priceUsd.toString(),
        priceHkd: price.priceHkd.toString(),
        marketCap: price.marketCap,
        volume24h: price.volume24h,
        percentChange24h: price.percentChange24h.toString(),
      });
    }
    
    // Check alerts
    await checkAlerts(prices);
    
  } catch (error) {
    console.error('Error checking prices:', error);
  }
}

async function checkAlerts(prices: any[]) {
  const activeAlerts = await db.select()
    .from(priceAlerts)
    .where(eq(priceAlerts.status, 'active'));
  
  for (const alert of activeAlerts) {
    const priceData = prices.find(p => p.coinId === alert.coinId);
    if (!priceData) continue;
    
    let shouldTrigger = false;
    const currentPrice = priceData.priceUsd;
    
    switch (alert.alertType) {
      case 'price_above':
        shouldTrigger = currentPrice > Number(alert.targetPrice);
        break;
      case 'price_below':
        shouldTrigger = currentPrice < Number(alert.targetPrice);
        break;
      case 'percent_increase':
        shouldTrigger = priceData.percentChange24h > Number(alert.percentChange);
        break;
      case 'percent_decrease':
        shouldTrigger = priceData.percentChange24h < -Number(alert.percentChange);
        break;
    }
    
    if (shouldTrigger && !alert.notificationSent) {
      await triggerAlert(alert, currentPrice, priceData);
    }
  }
}

async function triggerAlert(alert: any, currentPrice: number, priceData: any) {
  // Update alert status
  await db.update(priceAlerts)
    .set({
      status: 'triggered',
      triggeredAt: new Date(),
      triggerPrice: currentPrice.toString(),
      notificationSent: true,
    })
    .where(eq(priceAlerts.id, alert.id));
  
  // Get coin details
  const coin = await db.select()
    .from(trackedCoins)
    .where(eq(trackedCoins.coinId, alert.coinId))
    .limit(1);
  
  if (!coin.length) return;
  
  // Format message
  const message = formatAlertMessage(alert, coin[0], currentPrice, priceData);
  
  // Send notifications
  try {
    if (alert.notifySms) {
      await sendSMS(message);
    }
    if (alert.notifyWhatsapp) {
      await sendWhatsApp(message);
    }
  } catch (error) {
    console.error('Error sending notification:', error);
  }
}

function formatAlertMessage(alert: any, coin: any, currentPrice: number, priceData: any): string {
  const symbol = coin.symbol;
  const name = coin.name;
  const change24h = priceData.percentChange24h.toFixed(2);
  
  let message = `🚨 ${name} (${symbol}) Alert!\n\n`;
  
  switch (alert.alertType) {
    case 'price_above':
      message += `Price is now $${currentPrice.toFixed(2)}, above your target of $${Number(alert.targetPrice).toFixed(2)}`;
      break;
    case 'price_below':
      message += `Price is now $${currentPrice.toFixed(2)}, below your target of $${Number(alert.targetPrice).toFixed(2)}`;
      break;
    case 'percent_increase':
      message += `Price increased ${change24h}% in the last 24h (target: ${alert.percentChange}%)`;
      break;
    case 'percent_decrease':
      message += `Price decreased ${change24h}% in the last 24h (target: ${alert.percentChange}%)`;
      break;
  }
  
  message += `\n\nCurrent Price: $${currentPrice.toFixed(2)} USD`;
  message += `\n24h Change: ${change24h}%`;
  
  if (alert.label) {
    message += `\n\nNote: ${alert.label}`;
  }
  
  return message;
}
```

```tsx
function AddCoinDialog({ open, onOpenChange }: DialogProps) {
  const [searchQuery, setSearchQuery] = useState('');
  const [selectedCoin, setSelectedCoin] = useState(null);
  
  const { data: searchResults } = useQuery({
    queryKey: ['/api/admin/crypto/search', searchQuery],
    enabled: searchQuery.length > 2,
  });
  
  const addCoinMutation = useMutation({
    mutationFn: async (coinData: any) => {
      return apiRequest('POST', '/api/admin/crypto/coins', coinData);
    },
    onSuccess: () => {
      toast({ title: 'Coin added successfully' });
      onOpenChange(false);
    },
  });
  
  return (
    <Dialog open={open} onOpenChange={onOpenChange}>
      <DialogContent>
        <DialogHeader>
          <DialogTitle>Add Cryptocurrency</DialogTitle>
        </DialogHeader>
        
        <div className="space-y-4">
          <div>
            <Label>Search Coin</Label>
            <Input
              placeholder="Bitcoin, Ethereum, etc."
              value={searchQuery}
              onChange={(e) => setSearchQuery(e.target.value)}
            />
          </div>
          
          {searchResults && (
            <div className="max-h-60 overflow-y-auto space-y-2">
              {searchResults.map((coin: any) => (
                <div
                  key={coin.id}
                  className="p-3 border rounded-lg cursor-pointer hover:bg-muted"
                  onClick={() => setSelectedCoin(coin)}
                >
                  <div className="flex items-center gap-2">
                    <img src={coin.thumb} alt={coin.symbol} className="w-6 h-6" />
                    <div>
                      <div className="font-medium">{coin.name}</div>
                      <div className="text-sm text-muted-foreground">{coin.symbol}</div>
                    </div>
                  </div>
                </div>
              ))}
            </div>
          )}
          
          {selectedCoin && (
            <div className="p-4 bg-muted rounded-lg">
              <p className="font-medium">Selected: {selectedCoin.name}</p>
            </div>
          )}
        </div>
        
        <DialogFooter>
          <Button variant="outline" onClick={() => onOpenChange(false)}>Cancel</Button>
          <Button
            onClick={() => addCoinMutation.mutate(selectedCoin)}
            disabled={!selectedCoin}
          >
            Add Coin
          </Button>
        </DialogFooter>
      </DialogContent>
    </Dialog>
  );
}
```

## Part 7: Security & Best Practices

### Security Measures

1. **API Keys**:
   - Store in environment variables
   - Never commit to git
   - Encrypt Twilio tokens in database

2. **Rate Limiting**:
   - Implement max notifications per hour
   - Respect API rate limits (CoinGecko, News API)

3. **Error Handling**:
   - Graceful failures for API timeouts
   - Log errors without exposing sensitive data
   - Retry logic for failed notifications

4. **Data Validation**:
   - Validate phone numbers
   - Sanitize search queries
   - Validate price thresholds

### Performance Optimization

1. **Caching**:
   - Cache coin prices for 1 minute
   - Cache news articles for 1 hour

2. **Database Indexing**:
   - Index on coin_id and timestamp
   - Index on alert status

3. **Background Jobs**:
   - Use separate process for price monitoring
   - Implement job queues for heavy operations
